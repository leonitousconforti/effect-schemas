# effect-schemas

A collection of typed [Effect-TS](https://effect.website) schemas for common domain types.

## Modules

- **Finance** — `Bic`, `EthereumAddress`, `BitcoinAddress`
- **Geography** — `Latitude`, `Longitude`, `LatLon`, `PostalCode`, `AlphanumericGeocode`
- **Internet** — `Port`, `PortWithMaybeProtocol`, `MacAddress`, `IPv4`, `IPv6`, `IPv4Bigint`, `IPv6Bigint`, `Address`, `AddressBigint`, `IPv4CidrBlock`, `IPv6CidrBlock`, `CidrBlock`, `IPv4CidrBlockFromString`, `IPv6CidrBlockFromString`, `CidrBlockFromString`
- **Math** — `Operation`, `Operator`
- **Person** — `Sex`, `FirstName`, `MiddleName`, `LastName`, `Name`
- **String** — `isAscii`, `isAlphanumeric`, `isHexadecimal`, `isOctal`
- **Temperature** — `Kelvin`, `Celsius`, `Fahrenheit`, `KelvinFromString`, `CelsiusFromString`, `FahrenheitFromString`
